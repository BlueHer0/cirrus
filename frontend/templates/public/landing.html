Todo lo que necesitas

Una plataforma completa para gestionar tus comprobantes fiscales digitales.

📥

Descarga Automática

Descarga masiva de CFDIs desde el portal del SAT usando tu e.firma (FIEL). Sin intervención manual.

🔒

Almacenamiento Seguro

XMLs encriptados en MinIO con redundancia. Tu FIEL se almacena con cifrado Fernet. Nunca en texto plano.

📄

PDF y Excel

Genera PDFs profesionales con logo corporativo. Exporta en Excel o CSV con filtros avanzados.

🏢

Multi-Empresa

Gestiona múltiples RFCs desde un solo panel. Cada empresa con su FIEL, logo y configuración independiente.

🔑

API REST

API completa con autenticación por API Key. Integra Cirrus con tus sistemas contables y ERPs.

⏰

Programación

Programa descargas automáticas diarias, semanales o mensuales con Celery Beat.
